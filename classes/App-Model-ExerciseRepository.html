Exercise Repository.

Concrete infrastructure implementation for exercise data persistence. Implements ExerciseImporterPort (consumed by ImportExercisesUseCase), ExerciseLookupPort (consumed by ImportAttemptsUseCase), and ExerciseListReaderPort (consumed by ListExercisesUseCase) so that the dependency direction follows the Dependency Inversion Principle.
